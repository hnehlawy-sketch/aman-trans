🔧 الإعداد

تفعيل Firebase

استخدم Firestore بدل data.json المحلي

🔒 يُخزَّن مشفراً، لا يُعرض مجدداً. إذا كان مضبوطاً ستظهر "••• (set) •••"
📖 طريقة الإعداد
الخطوات:
1. اذهب إلى console.firebase.google.com
2. أنشئ مشروعاً → Firestore Database
3. Project Settings → Service Accounts
4. Generate New Private Key → حمّل JSON
5. الصق محتوى الـ JSON في الحقل أعلاه
6. اضغط اختبار ← مزامنة ← حفظ
Collections في Firestore:
users/ sessions/ notifications/ meta/settings
Firebase Spark (مجاني): 1GB storage، 50K reads/day
